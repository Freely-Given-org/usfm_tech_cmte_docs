== Summary

Description:: Deuterocanonical content. The marked content is relevant for publications which include Deuterocanonlical (DC) books. 
Syntax::
* *USFM:* `+\dc_{content}\dc*+`
* *USX:* `+<char style="dc">{content}</char>+`
Added:: 1.0
