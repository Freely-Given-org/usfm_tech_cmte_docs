== Summary

Description:: Embedded list entry.
- The variable `#` represents the level of indent.
Syntax::
- *USFM:* `+\lim#_text...+`
- *USX:* `+<para style="lim#">text...</para>+`
// tag::spec[]
Added:: 3.0
// end::spec[]

== Properties

StyleType:: Paragraph
OccursUnder:: `[ChapterContent], [Sidebar]`
TextType:: VerseText
TextProperties:: paragraph, publishable, vernacular
