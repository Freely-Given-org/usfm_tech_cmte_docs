== Summary

Description:: Indented paragraph with hanging indent.
* The variable `#` represents the level of overall paragraph indent.
Syntax::
* *USFM:* ``++\ph# ++``#__content__#
* *USX:* ``++<para style="ph#">++``#__content__#``++</para>++``
// tag::spec[]
Added:: 1.0
Deprecated:: 3.0
// end::spec[]
